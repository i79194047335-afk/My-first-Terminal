# Chart Briefing Log — автоматизированная система брифингов

## 2026-07-09/10 — Запуск системы

**Что сделано:**

### `pre_session_brief.py` — генератор брифингов
- 4 пары (EUR/USD, USD/JPY, AUD/USD, USD/CAD) × 3 сессии (Asia/London/NY)
- Автоопределение сессии по UTC-часу
- RSS-новости (Reuters, Bloomberg, FT, WSJ, CNBC) → фильтрация по валютам
- Технический контекст из market_memory.json (D1/H4 уровни, тренд, позиция)
- Экономический календарь из data_loaders/news_calendar.csv
- DeepSeek API — нарративный промпт в стиле аналитической записки
- Накопленный контекст между брифингами (`briefing_context.json`)
- Выход: `briefing.json` (полный) + `session_bias.json` (legacy для signal_engine.py)

### `server.py` — briefing_watcher
- Поллинг `briefing.json` каждые 5 сек
- Broadcast всем WebSocket-клиентам при изменении
- Push брифинга новому клиенту при подключении

### `index.html` — панель брифинга в терминале
- Кнопка **📋 Brief** в правом тулбаре (колонка 2)
- Overlay-панель поверх графика (`#briefPanel`, z-index: 2001)
- `overflow-y: auto` для длинного контента
- Отображение: повалютный bias → технический анализ → уровни → обоснование → сценарий → накопленный контекст → ожидаемые новости → риски → рекомендация
- Время составления в UTC+5
- Перерендер при смене символа
- Предупреждение при устаревании >12ч

### Cron
```
0  4 * * * — Pre-Asia  (04:00 UTC+5 = 23:00 UTC)
0 12 * * * — Pre-London (12:00 UTC+5 = 07:00 UTC)
0 17 * * * — Pre-NY    (17:00 UTC+5 = 12:00 UTC)
```

### Файлы
- `briefings/` — папка с логами брифингов
  - `BRIEFING_LOG.md` — исторический ручной лог (30 июня – 8 июля)
  - `CHART_BRIEFING_LOG.md` — этот файл
- `briefing.json` — текущий брифинг (runtime)
- `briefing_context.json` — накопленный контекст между сессиями (runtime)
- `briefing_cron.log` — лог cron-запусков
